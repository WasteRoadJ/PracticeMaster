操作步骤：

  1. 随意打开一个AI网站（DeepSeek、豆包等等）
  2. 上传原始题库文档（Word、文本）
  3. 复制下方提示词发给AI（转化效果不好请自行优化提示词）
  4. 将AI生成的题库复制到txt文本
  5. 将txt文本文件后缀名改为.csv
  6. 用WPS或微软office（可能会乱码）打开这个csv文件
  7. 将这个csv文件另存为excel的.xlsx文件
  8. 刷题助手导入这个.xlsx文件开始刷题
📊 数字版提示词（标准题库） 
你是一名题库格式转换专家。请将上传 Word/文本 题目严格按照以下规则转换为 CSV 格式，并以 **Markdown 代码块** 输出（用 ```csv 开头、``` 结尾），不要输出任何解释性文字。

【全局预处理（解决 OCR 错误）】
1. 将因 OCR 造成的奇怪换行合并成完整段落，单字或标点孤立行都应合并到上一行。
2. 删除首尾及连续空格，中文之间不要空格，英文单词间保留一个空格。
3. 纠正 OCR 常见错误：数字 0 与字母 O 按语境正确识别；英文逗号 , 改中文逗号 ，；英文括号 ( ) 改中文括号 （ ）；删除乱码符号   等。
4. 保证每道题是一个完整段落，不同题目之间用空行隔开。

【固定表头（14列，顺序不可变）】
题目内容,答案,题目类型,题型分类,题目图片,解析,选项1,选项2,选项3,选项4,选项5,选项6,选项7,选项8

【各题型转换规则】

1. 单选题
- 题目内容：删除末尾的答案标识（如 A、B、C）
- 答案：将选项字母转为数字 A→1, B→2, C→3, D→4, E→5, F→6, G→7, H→8
- 题目类型：single
- 题型分类：根据题目内容，智能归类
- 题目图片：留空（如题干中有则填入 base64 编码）
- 解析：留空（如有则填入）
- 选项：必须将所有选项文本按字母顺序完整填入 G-N 列（A→G列, B→H列, C→I列, D→J列, E→K列, F→L列, G→M列, H→N列）。如果选项是图片，将图片 base64 编码放入对应选项列。不足 8 个留空，超过 8 个只取前 8 个。绝对不许只填正确答案那一项。

2. 多选题
- 题目内容：删除末尾的答案标识（如 ABD、1,2,4）
- 答案：字母转数字，多个答案用英文逗号分隔，如 ABD → 1,2,4
- 题目类型：multiple
- 题型分类：同上分类规则
- 题目图片：同上分类规则
- 解析：同上分类规则
- 选项：同样必须将所有选项文本完整填入 G-N 列，规则同单选题。

3. 判断题
- 题目内容：删除“正确/错误”“对/错”等答案部分，只保留陈述句。
- 答案：正确填 1，错误填 2
- 题目类型：judge
- 题型分类：同上分类规则
- 题目图片：同上分类规则
- 解析：同上分类规则
- 选项：G 列必须填“正确”，H 列必须填“错误”，其余选项列全部留空。

4. 填空题
- 题目内容：保留完整题干，包括下划线 ___ 或括号（）
- 答案：标准答案，填入对应选项列的序号。将答案文本按顺序填入 G-N 列（一个空一列，从 G 列开始），然后在“答案”列填入对应的列序号。例如：G 列对应 1，H 列对应 2，以此类推。若有多个空，用英文逗号分隔（如 1,2）
- 题目类型：fill
- 题型分类：同上分类规则
- 题目图片：同上分类规则
- 解析：同上分类规则
- 选项：将答案文本按顺序填入 G-N 列，每列一个答案。不足 8 列留空，超过 8 个只取前 8 个。

5. 简答题
- 题目内容：保留完整题干
- 答案：直接填写参考答案或“略”
- 题目类型：essay
- 题型分类：同上分类规则
- 题目图片：同上分类规则
- 解析：同上分类规则
- 选项列全部留空

快速上手

  • 📥 导入题库：左侧菜单 📥→📂 导入本地题库，支持 .xlsx、.enc 或 URL。表格应含“题目内容、答案、题目类型”，选项列为“选项1～8”或数字/字母。
  • ✏️ 刷题模式：勾选/填写答案后提交，自动判对错并记录错题。答错时正确答案闪烁（可在设置中调整闪烁秒数，0=常亮）。
  • 📖 背题模式：直接显示正确答案与解析，适合快速记忆。
  • 🎲 顺序/随机：点击题目区右上角“顺序”按钮可切换随机模式，随机打乱题目顺序。
  • 🔥 错题强化：在错题库中开启（设置→错题强化学习次数），需连续答对指定次数才能移出错题，答错即显示解析。
  • 📝 模拟测试：其他→模拟测试，可自定义各题型数量、分值和限时，随机抽题，结束后显示得分与错题回顾。
  • 🔍 全文搜索：右侧导航上方输入框可按题号或题目内容模糊搜索。
  • ⭐ 收藏：点击“收藏”标记题目，收藏题目在导航栏显示金色边框，可导出收藏本。
  • 🤖 AI解析：点击“AI解析”调用在线接口，在设置中可更换 API 地址。
  • 📤 导出：支持导出完整题库（含进度、答题状态）、加密题库、错题本、收藏本及打印版。
  • 🔁 题库转化：其他→题库转化，支持 Word/文本→Excel 和 Excel→Word/文本 双向转化，内置提示词可直接复制到 DeepSeek、豆包等 AI 使用。⚠️ AI 转化结果需人工校对，不能保证百分百正确。
  • ⚡ 快捷键：数字键1-8选择选项，Enter提交，↑/↓切换题目，数字9随机跳转（输入框内无效）。

常见问题

  • 题库格式要求？ 推荐使用本工具导出的模板。表头为“题目内容、答案、题目类型、题型分类”等，选项用“选项1～8”。
  • 答案是字母（如A、ABD）能识别吗？ 能，A→1，B→2，多选、填空和判断题的各种写法（正确/错误、√/×）均可自动解析。
  • AI解析失效？ 在“设置”中修改 API 地址，将 {prompt} 作为占位符。
  • 错题强化次数如何修改？ “设置”→“错题强化学习次数”，最小值为1，默认2。
  • 选项是图片怎么办？ ①图片多：图片保存本地，在选项列路径引用；②图片少：可将图片转为 Base64 编码（格式：data:image/png;base64,...）粘贴到选项列，系统会自动识别为可放大图片。在线转换工具
  • 怎么清空进度？ “其他”→“重置进度”可清空所有答题记录和强化状态。
